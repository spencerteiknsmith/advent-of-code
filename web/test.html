this is a different view of an [Advent of Code leaderboard].
rather than just displaying stars, the time it took to solve each problem is shown

note that in keeping with the rate limit request tied to the endpoint this page scrapes from,
this page takes a minimum 15 of minutes between pulling updates from the Advent of Code servers.
as such, this page does not update in real time and will not always have the most recent results

for each day (up to the most recent any competitor has completed at least one star on), there are three columns:
    -S1: the time it took from the release of the puzzle to when the first star was earned
    -S1: the time it took from the release of the puzzle to when the second star was earned
    -Δt: S2 - S1, or the time it took to finish Part Two after finishing Part One

additionally, another sort option is presented.
'local' and 'stars' are the same as on [the official site] - click 'Ordering' to see how those are calculated.
the new option, 'dtlocal', is calculated much like 'local' with the highest ranking earning N points, etc.
however, the initial ranking is done once per day, and is ordered by lowest Δt
